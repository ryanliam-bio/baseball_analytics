Baseball Analytics Portfolio
This repository contains various data science projects focused on MLB analytics, player performance, and biomechanics.

Included Projects:
Bat Speed Analysis: Year-over-year changes in swing metrics.

Pitching Biomechanics: Correlation between jump testing and pitch velocity.

Tigers Release Points: Analysis of release point consistency vs LHH/RHH.

This site is built using Hugo and the Hugo Blox framework.
